# 🔬 HDO Selectivity Prediction using Machine Learning

## 📌 About the Project
This project applies Machine Learning techniques to predict the selectivity of compounds in hydrodeoxygenation (HDO) reactions.

The model is trained using experimental data, considering variables such as temperature, pressure, catalyst properties and reaction conditions.

---

## 🚀 Objectives

- Analyze the impact of operational variables
